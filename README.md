# 宝石命名赌局游戏

一款基于静态网页的宝石科普赌局游戏，玩家通过拖拽词条组合成"命名方案"，系统按乘积规则结算分数。

## 快速开始

### 方式一：使用 Live Server（推荐）

1. 在 VS Code 中安装 Live Server 扩展
2. 右键点击 `index.html` 文件
3. 选择 "Open with Live Server"
4. 浏览器会自动打开游戏页面

### 方式二：直接打开

1. 双击 `index.html` 文件
2. 浏览器会打开游戏页面（某些功能可能受限）

## 游戏玩法

1. **开始界面**：查看规则说明和排行榜，点击开始游戏
2. **选择词条**：从下方词条池拖拽词条到右侧倍率区（最多8个）
3. **调整筹码**：点击 ×2（翻倍）、÷2（减半）或 ALL IN（全押）
4. **开牌结算**：点击开牌按钮，词条依次翻转揭示倍率
5. **查看结果**：根据倍率乘积计算得分，查看科普说明
6. **继续游戏**：完成10题或筹码归零后，显示最终得分和排名

## 核心规则

- 基础筹码初始为10，每回合可调整
- 正确称谓（正倍率）：+1.0 ~ +2.5
- 混淆矿物（负倍率）：-0.5 ~ -2.0
- 误导名称（零倍率）：直接得0分
- 遇到0或负数倍率立即停止翻牌

## 部署说明

### GitHub Pages

1. 将项目推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 选择 main 分支，根目录
4. 点击 Save，等待部署完成
5. 访问 `https://<username>.github.io/<repo-name>`

### 其他静态托管

- Vercel：直接导入 GitHub 仓库
- Netlify：连接 GitHub 仓库，自动部署
- 阿里云/腾讯云 OSS：上传静态文件

## 技术栈

- **前端**：纯 HTML5 + CSS3 + JavaScript
- **存储**：localStorage（排行榜）
- **动画**：CSS Animations + JavaScript
- **数据**：JSON 文件（题库）

## 扩展建议

1. **新增题目**：直接编辑 `questions.json` 文件
2. **添加图片**：将图片放入 `images/` 目录，更新题库中的路径
3. **修改规则**：编辑 `script.js` 中的游戏逻辑
4. **美化界面**：修改 `styles.css` 中的样式
