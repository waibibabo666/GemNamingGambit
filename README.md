# 🎰 宝石命名赌局 · Gem Naming Gambit

> 一款宝石科普静态网页游戏。观察宝石图片，从词条池中拖拽正确的命名词条，乘积结算，在霓虹光芒中赢下筹码。

**🟡 这是电脑版** — 没有部署，可以下载，核心玩法可玩。

[![GitHub Pages](https://img.shields.io/badge/demo-GitHub%20Pages-gold)](https://waibibabo666.github.io/GemNamingGambit/)


---

## 游戏玩法

1. **观察宝石图片**，从词条池拖拽正确的命名词条到倍率区（最多 8 个）
2. **调整筹码** — ×2 翻倍 / ÷2 减半 / ALL IN 全押（每局限一次）
3. **开牌** — 词条逐一翻转揭示倍率，乘积结算
4. **正倍率加分**，**零倍率 Bust 归零**，**负倍率立即停牌扣分**
5. 共 **10 题**，筹码归零或完成全部题目后结算最终排名

## 技术栈

| 层 | 选型 |
|----|------|
| HTML | 纯 HTML5，四屏状态机 |
| CSS | 自定义属性体系 · `Noto Sans SC` + `Space Mono` 字体 · CRT 扫描线覆层 |
| JS | 原生 ES6，HTML5 Drag & Drop API，`async/await` 翻牌动画 |
| 存储 | `localStorage`（排行榜，最多 100 条） |
| 数据 | `questions.json` — 非开发人员可直接编辑扩展题库 |

**零依赖、零构建** — 打开即玩，部署只需上传静态文件。

## 扩展指南

- **加题**：编辑 `questions.json`，每条包含 `id / image / gemName / terms[]`（详见已有条目格式）
- **加图**：放入 `images/` 目录，文件名在题库中引用
- **改规则**：`script.js` 顶部变量控制回合数、初始筹码、彩蛋概率等
- **改样式**：`styles.css` 顶部 `:root` 自定义属性控制全套配色
